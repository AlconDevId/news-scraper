# Tech News Scraper 🔍
**Author:** Rizky Maulana | Alcon Dev  
**Language:** Python 3  

## What it does
Scrapes the top articles from Hacker News including:
- Article title
- Link
- Score (upvotes)
- Author
- Timestamp

Results are saved to a CSV file automatically.

## Requirements
```
pip install requests beautifulsoup4
```

## How to run
```
python scraper.py
```

## Output
Creates `tech_news.csv` with all scraped articles.
